Nota Metodológica: Correção de Dados

Fluxo de Ajuste

1. Problema Identificado

Sistema PHC gera automaticamente um orçamento adicional para cada factura emitida ao cliente HH Group.

2. Ajuste Aplicado

Facturas HH Group estimadas: 248 (78.3% de 317 facturas totais).

Porquê esta Correção?

  • O sistema PHC gera orçamentos duplicados para todas as faturas do HH Group.
  • Estes orçamentos são fantasmas — não representam oportunidades reais.
  • Incluir estes documentos reduz artificialmente a taxa de conversão da DIGITAL.
  • Com o ajuste, a conversão DIGITAL revela-se competitiva: 45.4%
  • Mesmo corrigindo, persiste o problema base: -75.4% volume vs IMACX
